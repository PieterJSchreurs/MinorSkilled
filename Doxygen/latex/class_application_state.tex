\doxysubsection{Detailed Description}
This is the base class for any \mbox{\hyperlink{class_application_state}{Application\+State}}. You can subclass it to implement a specific application state (e.\+g. logging in, in the lobby, playing a game, etc).

Each state has access to the finite state machine (fsm) it is a part of, so it can switch state and communicate.
