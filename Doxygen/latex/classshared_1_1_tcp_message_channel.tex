\doxysubsection{Detailed Description}
\mbox{\hyperlink{classshared_1_1_tcp_message_channel}{Tcp\+Message\+Channel}} is sort of a facade around a Tcp\+Client, Packets \& the Stream\+Util class. It abstracts communication to a single bidirectional channel which you can use to pass \mbox{\hyperlink{classshared_1_1_a_serializable}{ASerializable}} objects back and forth, and check to see whether everything is still peachy where the underlying connection is concerned.

Basically after the initial setup, \textquotesingle{}all\textquotesingle{} you have to worry about is having a channel and being able to push objects through it.

If you want to implement your own serialization mechanism, this is the place to do it.
